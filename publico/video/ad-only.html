Incontent ad-only com IMA direto

Este exemplo não usa Video.js nem dummy MP4 no modo incontent. Usa o Google IMA SDK diretamente, pede a tag VAST e mostra apenas a publicidade. Se houver erro, no-fill ou timeout, substitui pelo HTML fallback. Quando a publicidade termina, o slot não colapsa.

Para mudares o HTML alternativo, edita o conteúdo dentro de <template id="fallback-incontent">. Este fallback só é usado em timeout, erro ou no-fill, não quando o anúncio termina normalmente.
