Video.js + Google IMA Publisher Player

Esta versão tem dois motores: Video.js + videojs-ima para preroll com conteúdo, e IMA SDK direto para incontent/ad-only sem dummy MP4.

Começa por testar o primeiro player. Ele já tem um vídeo de exemplo estilo sci-fi/space, parecido com o demo anterior que tinhas gostado, para veres imediatamente a sequência: publicidade GAM → conteúdo de vídeo.

Para produção, muda apenas data-video-src, data-video-type e data-poster. O teu data-ad-tag-url já está colocado com os parâmetros VAST programmatic/AdX fixos na própria tag.

Modo 1: preroll + conteúdo de vídeo

Este exemplo carrega uma playlist sequencial a partir de playlist.json. Antes do primeiro vídeo há preroll; quando cada vídeo termina, o player passa para o seguinte e faz nova chamada VAST com correlator novo.

Área de scroll para teste do floating

Depois de a publicidade começar, faz scroll para baixo até o player original sair totalmente do viewport. Se data-ad-floating="true" estiver activo e houver publicidade em reprodução, o player pequeno deve aparecer no canto inferior direito.

Continuação do artigo

Este bloco serve apenas para simular uma página longa, semelhante a uma notícia real. Ao regressares à zona original do player, o floating deve desaparecer imediatamente.

Mais conteúdo de teste

Quando a publicidade terminar, o player pequeno deve desaparecer. Se fechares o player pequeno, a publicidade fica em pausa no placement principal até o utilizador carregar novamente em play.

Modo 2: incontent/ad-only sem conteúdo

Este exemplo usa IMA SDK direto, sem Video.js e sem dummy MP4. Se houver anúncio, mostra a publicidade e mantém o slot após terminar. Se houver erro, no-fill ou timeout, chama o HTML fallback.
